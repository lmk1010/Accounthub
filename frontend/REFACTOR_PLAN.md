## 开发规范

### 样式规范
1. **直接复用原 CSS**: 尽可能直接使用原 `base.css` 和组件 CSS
2. **保持类名一致**: 使用与原 HTML 相同的类名
3. **不修改样式**: 除非必要，不修改原样式代码

### 组件规范
1. **功能对等**: 每个 React 组件必须实现原 HTML 的所有功能
2. **交互一致**: 所有用户交互必须与原界面一致
3. **数据流清晰**: 使用 React 状态管理，但保持逻辑与原代码对应

### 测试规范
1. **视觉对比**: 每完成一个页面，必须与原界面进行视觉对比
2. **功能验证**: 所有功能必须经过测试验证
3. **响应式测试**: 在不同屏幕尺寸下测试
